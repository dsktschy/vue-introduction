## Vue CLIとは
* Vue.jsを使用するアプリケーションを簡単に構築するツール
* ターミナル上でコマンドによって操作する

### メリット
* Webpackの詳解と設定作業をスキップできる
* 様々な形式のアプリケーションを構築可能
  * 後述のフレームワークNuxt.jsもVue CLIを用いて構築できます。
